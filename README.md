## Как просмотреть отчет Power BI

Для полноценного просмотра содержимого файла `.pbix` вам потребуется установленное приложение **Power BI Desktop**.

1. Кликните на название файла в списке (например, `report.pbix`).
2. На открывшейся странице нажмите кнопку **View raw** (Посмотреть оригинал). 
   *Файл автоматически загрузится на ваш компьютер.*
3. Откройте скачанный файл через **Power BI Desktop**.
